# Greyscale AI Static Website

Clean static rebuild of the Greyscale AI development website for GitHub Pages.

## Structure

- `index.html` is the home page.
- `solutions/`, `markets_and_applications/`, `careers/`, `contact/`, and `news/` preserve the current public site routes.
- `assets/css/styles.css` contains all site styling.
- `assets/js/main.js` contains mobile navigation and the home hero image rotation.
- `assets/img/` contains downloaded brand, product, market, solution, contact, and news assets from the current site.

## GitHub Pages

This repo is intended to be served as a development site from GitHub Pages:

- Source: branch `main`
- Folder: `/root`
- Expected development URL: `https://greyscaleai521.github.io/draft-website-dev/`

Do not add a `CNAME` file for `www.greyscaleai.com` while this remains a development site.

## Indexing

The pages include `noindex, nofollow` meta tags, and `robots.txt` disallows crawling. Remove those before production launch.

## Notes From Inventory

The current live site is generated by a no-code platform and includes runtime JavaScript, service worker code, tracking scripts, generated CSS, duplicated navigation markup, and placeholder slide content. This rebuild keeps the public information architecture and useful visual assets while replacing the runtime with hand-authored HTML, CSS, and a small JavaScript file.
